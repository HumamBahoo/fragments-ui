You can append the desired extension to which type you want your fragment to be converted to. For example: .txt, .md, etc...

Current Supported conversion types:

  • text/plain => text/plain (.txt)
  • text/markdown => text/html (.html)
  • text/html => text/plain (.txt)
  • application/json => text/plain (.txt)
